# MailerLite Email Styling Guide

Use these brand settings when creating emails in MailerLite's drag-and-drop editor.

## Brand Colors

| Element | Color | Hex Code |
|---------|-------|----------|
| **Primary/Buttons** | Rose Pink | `#d63384` |
| **Button Hover** | Dark Rose | `#b52a6f` |
| **Background** | Soft White | `#fdfdf9` |
| **Content Area** | Pure White | `#ffffff` |
| **Header/Footer BG** | Light Pink | `#fef7fb` |
| **Body Text** | Soft Black | `#1a1a1a` |
| **Secondary Text** | Warm Gray | `#4a4a4a` |
| **Dividers** | Pale Pink | `#fdf2f7` |

## Typography

- **Headings**: Use sans-serif (Arial/Helvetica in MailerLite)
- **Body text**: Use serif if available, otherwise default
- **Heading size**: 28-32px
- **Body size**: 17-18px
- **Line height**: 1.7 (or "relaxed" spacing)

## Layout Settings

- **Content width**: 600px max
- **Padding**: 40px on sides
- **Border radius**: 8px on containers
- **Button padding**: 16px vertical, 32px horizontal
- **Button border radius**: 6px
