Update the scroll event handler in the <script> section:
```javascript
// Navigation scroll effect with hide/show
let lastScroll = 0;
const navbar = document.getElementById('navbar');
const scrollThreshold = 5; // Minimum scroll delta to trigger hide/show

window.addEventListener('scroll', () => {
  const currentScroll = window.pageYOffset;
  const scrollDelta = currentScroll - lastScroll;

  // Add shadow when scrolled past 50px
  if (currentScroll > 50) {
    navbar?.classList.add('scrolled');
  } else {
    navbar?.classList.remove('scrolled');
  }

  // Hide/show based on scroll direction (only after initial 100px)
  if (currentScroll > 100) {
    if (scrollDelta > scrollThreshold) {
      // Scrolling down - hide nav
      navbar?.classList.add('hidden');
    } else if (scrollDelta < -scrollThreshold) {
      // Scrolling up - show nav
      navbar?.classList.remove('hidden');
    }
  } else {
    // Always show nav at top of page
    navbar?.classList.remove('hidden');
  }

  lastScroll = currentScroll;
});
```

Key behaviors:
- Nav always visible at top of page (first 100px)
- Scrolling down hides nav (slide up off screen)
- Scrolling up shows nav (slide back down)
- Small scroll threshold prevents jitter on minor scroll movements
- Maintains existing shadow effect
