I've created three HTML email templates in the `email-templates/` directory:
- `welcome.html` - For new subscriber welcome automation
- `book-release.html` - For book launch announcement campaign
- `newsletter.html` - For regular newsletter campaigns

**To upload and configure in MailerLite:**

1. **Create Welcome Automation:**
   - Go to MailerLite dashboard → Automations → Create automation
   - Trigger: "When subscriber joins a group" → Select "Book Updates" group
   - Add step: "Send email"
   - In the email editor, switch to "Custom HTML" mode
   - Copy/paste contents of `email-templates/welcome.html`
   - Set subject line: "Welcome to the Living to Die Journey"
   - Set from name: "Diane Melton" (or preferred sender name)
   - Activate the automation

2. **Save Book Release as Draft Campaign:**
   - Go to Campaigns → Create campaign
   - Name: "Book Release Announcement" (save as draft for later)
   - In editor, use "Custom HTML" and paste `book-release.html`
   - Leave as draft until book is released

3. **Save Newsletter as Template:**
   - Go to Templates → Create template
   - Name: "Living to Die Newsletter"
   - Use "Custom HTML" and paste `newsletter.html`
   - Save for future newsletter campaigns

**Test the welcome automation:**
- Subscribe with a test email to verify welcome email arrives correctly
- Check formatting renders properly in your email client
