```typescript
const authCookie = Astro.cookies.get('admin_auth');
const isAuthenticated = authCookie?.value === 'true';

if (!isAuthenticated) {
  return Astro.redirect('/admin');
}
```

- Match existing admin styling (system-ui font, #f5f5f5 background, white cards)
- Primary button color: #8B3A62 (matches existing)

- Clear visual feedback on send
